## Orchestration pattern

- Quality check agent: verify every profile is complete, consistent, and aligned with the Davey Capital strategy.

## Intake checklist
For each agent, capture:
- role name
- mission
- inputs
- outputs
- operating cadence
- guardrails
- dependencies
- success criteria
- escalation path
