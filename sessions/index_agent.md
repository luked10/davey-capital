# sessions index agent

This folder is the daily source-of-truth for trading workflow checkpoints and the audit trail.

## Checkpoints

- pre_market_agent.md
- planning_agent.md
- execution_agent.md
- midday_agent.md
- close_agent.md

## Audit templates

- decision_log_agent.md
- fill_report_agent.md

## Usage

1. Start the day in pre_market_agent.md.
2. Capture live decisions in decision_log_agent.md.
3. Record routed fills in fill_report_agent.md.
4. Close the loop in close_agent.md.
