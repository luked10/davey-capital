# sessions index boi

This folder is the daily source-of-truth for trading workflow checkpoints and the audit trail.

## Checkpoints

- pre_market_boi.md
- planning_boi.md
- execution_boi.md
- midday_boi.md
- close_boi.md
- politician_ptr_alert_boi.md

## Audit templates

- decision_log_boi.md
- fill_report_boi.md

## Usage

1. Start the day in pre_market_boi.md.
2. Capture live decisions in decision_log_boi.md.
3. Record routed fills in fill_report_boi.md.
4. Write PTR cluster alerts into politician_ptr_alert_boi.md when the monitor trips.
5. Close the loop in close_boi.md.
